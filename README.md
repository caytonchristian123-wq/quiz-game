1. Frontend muna (recommended for beginners)

Gawin muna:

UI
layout
forms
buttons
pages

Example:

login page
employee form
sidebar
dashboard

Why?

mas madaling makita progress
mabilis mag-design
easier debugging
pwede ka muna gumamit ng fake data

After gumana UI:
➡️ saka ikokonekta sa backend/database.

Ito magandang approach lalo na sa ginagawa mong systems.

2. Backend muna

Ginagawa ito kapag:

database-heavy project
API muna priority
maraming business logic

Example:

authentication
payment system
inventory logic
3. Full-stack workflow (most common)

Usually ganito:

Design UI (frontend)
Create database
Build backend/API
Connect frontend ↔ backend
Testing

Example sa milk tea system mo:

Frontend
flavor buttons
quantity
add-ons
receipt UI
Backend
save order to SQLite/SQL
compute totals
generate OrderID
Connect
button click → send data to database
